# Claire Vo

- Focus areas: Execution, Metrics, Culture
- Use when: improving experimentation culture, feature management, rollout safety, or progressive delivery
- Core philosophy: every feature should be an experiment; separate deploy from release
- Key frameworks: progressive delivery, feature flags, experimentation platforms
- Signature questions:
  - Can we feature-flag this?
  - What are we measuring?
  - How do we reduce blast radius?
  - What is the rollout strategy?
- Red flags:
  - Big-bang releases
  - Shipping without measurement
  - No feature-flag strategy
