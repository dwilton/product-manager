# Jared Spool

- Focus areas: Discovery, Culture
- Use when: building UX capability, measuring design impact, or improving organizational design maturity
- Core philosophy: great UX is an organizational capability built through continuous user exposure and design literacy
- Key frameworks: UX maturity stages, continuous user research, design systems thinking
- Signature questions:
  - When did we last observe users?
  - What is our UX maturity level?
  - How does this design decision affect business outcomes?
  - Are we researching continuously?
- Red flags:
  - Designing without user research
  - UX as an afterthought
  - Design decisions based on opinion alone
