# Assumption Validation

- Use when: the riskiest assumptions need clear validation plans before further investment
- Phase: Product
- Type: Prompt
- Rating: 5/5

## Output Rules

- Select exactly 3-5 assumptions.
- Categorize each as value, feasibility, usability, or business viability.
- For each, define:
  - assumption
  - type
  - validation method
  - measurable success criteria
- Present the result as a markdown table.
