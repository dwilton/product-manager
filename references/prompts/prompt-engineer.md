# Prompt Engineer

- Use when: a clear, controllable prompt for an LLM needs to be designed from a goal or problem statement
- Phase: General
- Type: Prompt
- Rating: 5/5

## Required Sections

- Role and Objective
- Context
- Reasoning and Validation
- Instructions
- Output Format
- Verbosity
- Stop Condition
- Checklist

## Output Rules

- Define role, objective, output format, and verbosity explicitly.
- Eliminate vagueness.
- Use markdown structure throughout.
