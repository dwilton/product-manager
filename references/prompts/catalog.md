# Prompt Catalog

Use this file to find the closest matching prompt. Then open the specific prompt file if one exists.

## Discovery And Problem Framing

- `Problem Framing`: align on a clear problem statement, evidence, impact, users, HMWs, and success metrics
- `Opportunity Reframing`: turn HMW problem statements into opportunity-focused prompts
- `Opportunity Map`: cluster HMWs into 3-6 opportunity areas
- `Solution Ideation`: generate sketches, brainwriting, and lightning demos
- `Assumptions Map`: plot assumptions by evidence and importance
- `Assumption Validation`: define validation methods and success criteria for the riskiest assumptions
- `Persona`: create a research-grounded persona
- `User Journey Map`: map stages, actions, thoughts, pain points, and opportunities
- `Usability Testing`: observe real users completing tasks
- `Customer Interview Script`: prepare open-ended discovery questions
