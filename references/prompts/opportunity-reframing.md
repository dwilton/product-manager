# Opportunity Reframing

- Use when: HMW problem statements need to become opportunity-focused prompts
- Phase: Product
- Type: Prompt
- Rating: 5/5

## Output Rules

- Reframe each HMW into open-ended opportunity prompts.
- Favor phrasing like:
  - `What opportunities exist to...`
  - `In what ways could we...`
  - `How can we leverage...`
- Present results in a markdown table with original HMW and reframed prompts.
