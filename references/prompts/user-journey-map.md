# User Journey Map

- Use when: a persona and job-to-be-done need a structured journey map
- Phase: Product
- Type: Prompt
- Rating: 5/5

## Fixed Columns

- Stage
- User actions
- Goals and experiences
- Feelings and thoughts
- Pain Points
- Opportunities

## Output Rules

- Propose sensible stages if the user does not provide them.
- Return a markdown table only.
