# Persona

- Use when: a target user needs to be defined for discovery, design, or prioritization
- Phase: Product
- Type: Prompt
- Rating: 5/5

## Output Rules

- Create one persona only unless asked for more.
- Include:
  - persona name and archetype
  - a 2-3 sentence narrative
  - a table covering demographics, role, goals, behaviors, pain points, motivations, tools, and scenarios
- Use placeholders where research is missing rather than inventing details.
