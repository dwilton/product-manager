# Opportunity Map

- Use when: opportunity-focused HMWs need clustering into 3-6 opportunity areas
- Phase: Product
- Type: Prompt
- Rating: 5/5

## Output Rules

- Restate the overall challenge first.
- Create 3-6 opportunity areas.
- Give each area a short title and 2-3 sentence description.
- Map each HMW into at least one area.
- Do not generate solution ideas here.
