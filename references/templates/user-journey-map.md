# User Journey Map



A **User Journey Map** visualizes the end-to-end experience a user has when interacting with your product or service. It captures actions, thoughts, emotions, pain points, and opportunities at each stage.

Use this template to build empathy, identify friction points, and prioritize improvements based on where users struggle or succeed. Base it on real customer research — interviews, observations, analytics, and support data.
