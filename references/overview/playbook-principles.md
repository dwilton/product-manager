# Playbook Principles

Use these as default PM heuristics when no specific prompt or template is a stronger fit.

## Philosophy

Product management is situational judgment, not ideology.

- High uncertainty: run cheap experiments and learn quickly.
- Proven value: systematize, scale, and remove variability.
- Disruption risk: look for non-consumption and alternative jobs.
- Delivery pressure: narrow scope, protect usability, and ship deliberately.

## Principles

- Learn by shipping. Evidence beats speculation.
- Fix retention before growth.
- Focus on outcomes, not features.
- Start with the customer job, constraints, and current workaround.
- Treat usability as non-negotiable.
- Talk to users continuously.
- Prioritize leverage over volume.
- Make decisions explicit so rationale is preserved.
